# Docs

Este directorio existe para contener recursos de apoyo al proyecto
Nova Mint / Singapura que no pertenecen directamente al runtime... por ahora.

No se garantiza que el contenido esté actualizado,
completo o siquiera sea útil.

---

## Contenido

### Imágenes

- Recursos visuales usados en el README principal.
- Principalmente banners, capturas o material ilustrativo.
- No representan necesariamente el estado actual del sistema.

---

### Addons Snippets

Ejemplos simples de addons / snippets para el sistema Mint.

Incluye:

- `ClockDesk.js`  
  Ejemplo básico de widget de reloj.

- `ExampleApp.js`  
  Plantilla mínima para probar integración de apps.

- `HelloWorld.js`  
  Ejemplo trivial. Existe porque todo proyecto necesita uno.

Estos archivos sirven como referencia inicial.
No son APIs estables ni contratos de compatibilidad.

Los snippets son ejemplos de cómo usar la API interna de Mint,
pero no se asegura que funcionen en un futuro ni que sean actualizados.
